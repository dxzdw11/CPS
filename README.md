# Speed ​​Test | CPS & Spacebar Counter

A lightweight, modern, and responsive web application to test your reflexes by measuring **CPS (Clicks Per Second)** and typing speed on the **Spacebar**.

Developed entirely in native code (HTML5, CSS3, Vanilla JS), the app is structured in a single file and is 100% optimized for free hosting on **GitHub Pages**.

---

## 🚀 Main Features

* **Dual Mode:** Instantly switch between the mouse test (CPS) and the spacebar test.
* **Flexible Timer:** Choose the challenge duration from 1, 5, 10, 30, or 60 seconds.
* **Anti-Cheat System:** Integrated keyboard event control prevents you from accumulating points simply by holding down the spacebar.
* **Fluid Interface:** The timer updates every 50 milliseconds to display real-time CPS with maximum precision.
* **Modern Design:** Minimal, eye-friendly, and fully responsive dark theme (Dark Mode).
* **Zero Dependencies:** No heavy frameworks or external libraries. Just pure, lightweight, and lightning-fast code.
